# Tute-Dude-Ds-Assignment-6
# Sales Forecasting Case Study

This project demonstrates sales prediction using machine learning techniques on a retail sales dataset.

## Project Workflow

### 1. Data Exploration
- Loaded the dataset using Pandas
- Inspected:
  - Dataset structure
  - Data types
  - Missing values

### 2. Data Preprocessing
- Handled missing values in:
  - Item_Weight
  - Outlet_Size
- Encoded categorical variables using Label Encoding

### 3. Train-Test Split
- Divided the dataset into:
  - Training data
  - Testing data

### 4. Sales Prediction Model
- Implemented Linear Regression for sales forecasting
- Trained the model using multiple product and outlet-related features

### 5. Model Evaluation
- Evaluated model performance using:
  - Mean Absolute Error (MAE)
  - Root Mean Squared Error (RMSE)
  - R² Score

### 6. Visualization
- Visualized Actual vs Predicted sales using Matplotlib

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
